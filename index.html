Responsive Web Design Intro

Responsive web design is building a web site that looks good and works well on all browsers and all devices. Users have a variety of devices and screen sizes. Responsively designed web sites appear readable, easy to use, and attractive no matter the size of the screen it’s viewed from. If we don’t design sites considering these differences in viewers we will lose customers. Research shows people prefer to shop from well-designed responsive web sites. Some issues considered important to RWD are: fluid grids, flexible images, and media queries. Grids can adapt to screen size, shifting columns to fit appropriately. A 3 column layout can move to a 2 column layout. For smaller screen widths images should resize appropriately without becoming distorted or overcoming the page. Use media queries to discern size of screen. A first rule of thumb when designing web sites is to start with a design for a smaller screened device (i.e. mobile phone) then adapt for a larger screen.
